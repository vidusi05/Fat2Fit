# Fat2Fit Website Media Assets

Place your gym media files in this folder. The website is configured to look for the following filenames:

### Facilities
1. `cardio.jpg` - Image showing the cardio equipment / treadmill zone.
2. `strength.jpg` - Image showing the free weights and strength training zone.
3. `studio.jpg` - Image showing the yoga, pilates, or group class studio.
4. `general.jpg` - General gym floor overview image.

### Trainers
1. `trainer1.jpg` - Profile photo for Trainer 1 (e.g., Alex Carter).
2. `trainer2.jpg` - Profile photo for Trainer 2 (e.g., Sarah Jenkins).
3. `trainer3.jpg` - Profile photo for Trainer 3 (e.g., Marcus Vance).

### Hero Banner
1. `hero.jpg` - Large, high-resolution header/hero background image.

*Note: If no images are placed in this folder, the website will automatically display elegant CSS gradients as fallbacks so that the site remains functional and looks visually polished.*
